# 💊 Smart Medical Store Management System

## 📌 Description
The Smart Medical Store Management System is a Java-based application designed to manage the daily operations of a medical store. It helps in handling medicines, sales, and purchases efficiently while reducing human errors. The system is useful for store admins to maintain records and improve workflow.
