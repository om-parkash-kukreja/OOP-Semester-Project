# 💊 Smart Medical Store Management System

## 📌 Description
The Smart Medical Store Management System is a Java-based application designed to manage the daily operations of a medical store. It helps in handling medicines, sales, and purchases efficiently while reducing human errors. The system is useful for store admins to maintain records and improve workflow.

## ⚙️ Features / Modules
- Admin control for adding, updating, viewing, and deleting medicines  
- Medicine Sales Management  
- Medicine Purchase Management  
- Search medicines by name, ID, and company  
- Reduced human errors and improved efficiency

## ▶️ How to Run
1. Install Java (JDK 8 or above)  
2. Install MySQL and create a database  
3. Import the provided `.sql` file into MySQL
4. Open file in any text editor and compile on CMD 
5. Make sure database connection settings (username/password) are correct in the code
6. must Extract zip of of DB Connector and set class path of .jar file
7. mentioned commands should be used to compile and run
8. Compile -> javac -cp ".;mysql-connector-j-9.6.0.jar" smart.java
9. Run -> java -cp ".;mysql-connector-j-9.6.0.jar" smart
